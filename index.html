Green Eats – Full Stack TALL PHP Project

Type: Individual Full Stack Project

Green Eats is a dynamic and feature-rich fruit store website built using the TALL stack with PHP and Alpine.js. The site supports two distinct roles — Admin and User — each with its own interface and functionality.

What I Built:
– Designed and developed both the user and admin panels
– Admins can manage orders, add new fruits for sale, and handle registered users
– Implemented complete sign up / sign in functionality, along with forgot password recovery
– Added email notification system for password resets and account actions
– Created dynamic pages and conditional layouts based on user roles
– Built full CRUD functionality with real-time filter and search features
– All database operations and logic handled within PHP using the TALL architecture

Highlights:
This project gave me deep experience in full stack PHP development using the TALL stack approach. From building secure authentication to managing role-based access and real-time functionality, I implemented everything myself — including design, database, and user flow. It was a powerful hands-on learning experience in dynamic, scalable web app development.

PHP Tall Development - Training Session

Training Session Platform – TALL Stack PHP Project

Type: Team Full Stack Project (6 Members)

This project was a fully functional university platform built using the TALL stack (Tailwind, Alpine.js, Laravel, Livewire) and PHP. It supported multiple user roles including Student, Coach, Coordinator, and Admin — each with custom dashboards and permissions.

Key Features:
– Automatic role assignment based on email format (e.g., rnumber@student.thomasmore gets Student role)
– Email verification system on registration
– Role-based access control secured via middleware
– Assignment tracking, progress visualization, and secure data flow between roles
– Designed to be user-friendly and responsive for educational use

Team Collaboration:
We followed the Scrum methodology and used Jira for sprint planning, issue tracking, and workflow management. Working with a team of 6, we supported each other across tasks — operating like a well-oiled machine. I learned not just about advanced development, but also about agile teamwork and effective communication.

Highlights:
This project taught me the importance of structured collaboration, clean role separation, and building scalable educational platforms. From backend logic to front-end interactions, we worked together to build something impactful and well-architected.
